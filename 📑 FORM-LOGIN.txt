📋 Template Prompt Super Detail: Replikasi Pixel-Perfect LoginPage.tsx
Bertindaklah sebagai Senior Frontend Engineer. Tugasmu adalah membuat komponen halaman login yang 100% presisi (pixel-perfect) bernama LoginPage.tsx. Gunakan React, Tailwind CSS v4, dan komponen dari shadcn/ui.

1. Bagian Import (Wajib Sama Persis):
React: import { useState } from 'react';
Router: import { Link } from 'react-router-dom';
Ikon: import { Eye, EyeOff } from 'lucide-react';
Shadcn UI: import { Button } from '@/components/ui/button'; dan import { Input } from '@/components/ui/input';
Aset Logo: import logo from '@/assets/logo.svg'; (Ini krusial, jangan gunakan <svg> inline, panggil variabel logo ini di tag <img>).

2. State & Logika (Fungsionalitas):
Buat const [showPassword, setShowPassword] = useState(false);
Buat const [isError, setIsError] = useState(false);
Buat fungsi handleSubmit (dengan parameter e: React.FormEvent) yang menjalankan e.preventDefault() dan men-toggle nilai isError. Panggil fungsi ini pada properti onSubmit di dalam tag <form>.

3. Struktur Layout & Wrapper Utama:
Wrapper terluar: <div className='flex flex-col items-center justify-center min-h-screen w-full'>
Container form di dalamnya: <div className='w-full max-w-[400px] flex flex-col'>

4. Detail Elemen (Styling Spesifik):
Brand/Logo: Dibungkus div dengan class flex items-center justify-start gap-2 mb-5. Di dalamnya terdapat <img> dengan src={logo} ukuran w-[33px] h-[33px], disusul teks "Booky" dengan class font-bold text-[25.14px] leading-[33px] text-neutral-950.
Judul & Subjudul: Jarak bawah mb-[20px]. Teks "Login" menggunakan text-display-sm font-bold text-neutral-950. Subjudul "Sign in to manage..." menggunakan text-md font-semibold text-neutral-700.
Form: Jarak antar elemen <form> adalah gap-[16px].
Input Email:
Label: text-sm font-bold text-neutral-950.
Input prop: type='email', placeholder=''.
Class dinamis: Base class adalah rounded-xl px-[16px] py-[8px] h-auto. Jika isError true: gunakan border-[#EE1D52] text-neutral-950 focus-visible:ring-[#EE1D52]. Jika false: border-neutral-200 text-neutral-950.
Input Password:
Dibungkus div dengan posisi relative.
Input prop: Toggle tipe antara 'text' dan 'password', placeholder=''. Class dinamis sama dengan email, tambahkan pr-12.
Tombol Mata: absolute right-4 top-1/2 -translate-y-1/2 text-neutral-500 hover:text-neutral-700 transition-colors. Tampilkan EyeOff jika showPassword true, dan Eye jika false (ukuran 20).
Text Helper (Error): Tampilkan <span> bertuliskan "Text Helper" dengan class text-sm font-medium text-[#EE1D52] di bawah masing-masing input HANYA jika isError true.
Tombol Login: Tombol tipe submit, lebar penuh, class: w-full bg-primary-dark hover:bg-primary-dark/90 text-white rounded-full h-auto py-[10px] font-semibold text-md transition-all mt-1.

Footer Link: Teks di tengah. "Don't have an account?" (dengan class text-md font-semibold text-neutral-950) lalu tag <Link> ke /register (dengan class text-md font-semibold text-primary-dark hover:underline).
Tulis kode seutuhnya dari awal hingga akhir tanpa ada bagian yang disingkat atau dipotong.
